# MidnightEmber 🌙🔥

MidnightEmber is a premium fine-dining landing page showcasing flame-crafted contemporary cuisine in an intimate Manhattan setting. Built with Next.js 15, TypeScript, and Tailwind CSS.

## ✨ Features

- **Modern Stack**: Built with Next.js 15 and React 19 for peak performance.
- **Flame-Crafted Design**: A high-end, dark-themed aesthetic tailored for a premium restaurant brand.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop viewing.
- **Interactive Sections**:
  - Hero Section with dynamic visuals.
  - Story Section detailing the culinary philosophy.
  - Signature Dishes showcase.
  - Full Menu display.
  - Testimonials from guests.
  - Location & Reservation info.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Logic**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: Framer Motion / ScrollReveal
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🏁 Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Subham234s/Midnightember-Restaurant_2.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:4028](http://localhost:4028) in your browser.

## 📦 Available Scripts

- `npm run dev`: Starts the development server on port 4028.
- `npm run build`: Creates an optimized production build.
- `npm run serve`: Starts the production server.
- `npm run lint`: Checks for code quality issues.
- `npm run format`: Formats code using Prettier.

## 📱 Deployment

This project is optimized for deployment on **Vercel**.

1. Push your code to GitHub.
2. Connect your GitHub repository to Vercel.
